# Telegram Device Watcher

Home Assistant custom integration that notifies via Telegram when selected entities
become `unavailable` or `unknown`.

## Features
- UI configuration (no YAML)
- Select any entities (not only Zigbee)
- Telegram notifications
- Debounce (no spam)

## Installation (HACS)
1. HACS → Integrations → Custom repositories
2. Add: https://github.com/gresuk/telegram_device_watcher
3. Category: Integration
4. Install
5. Restart Home Assistant

## Usage
Settings → Integrations → Add Integration → Telegram Device Watcher
